Nesta página encontra-se as instruções de como utilizar este sistema.

Este sistema tem o propósito de receber e armazenar dados de vários dispositivos, sobretudo os relacionados à internet das coisas (IOT).

Primeiramente voce pode fazer o seu cadastro no sistema para poder usa-lo.

Após o cadastro temos varias opções na página principal:

Dashboard:

Esta é a página principal, nela temos a vizualizaçao da lista de links configurados.

Configuração dos links:

Nesta página é possivel fazer a configuração para os links da API que serão usados pelos dispositivos para o envio de dados.
É necessario adicionar apenas um nome para o link, e então será gerada uma chave aleatoria, ela sera usada para referenciar os dados dos dispositivos que a usar.

Graficos:

Aqui temos os graficos gerados com os dados dos links, inicialmente nos limitamos apenas à graficos de temperatura, mas em futuras versoes haverá uma forma de adicionar outros tipos de vizualização para dados mais gerais.
