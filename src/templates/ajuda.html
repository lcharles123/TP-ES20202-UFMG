Nesta página encontra-se as instruções de como utilizar este sistema.

Este sistema tem o propósito de receber e armazenar dados de vários dispositivos, sobretudo os relacionados à internet das coisas (IOT).

Primeiramente voce pode fazer o seu cadastro no sistema para poder usa-lo.

Após o cadastro temos varias opções na página principal:

Dashboard:

Esta é a página principal, nela temos a vizualizaçao da lista de links configurados e os dados recebidos.

Configuração dos links:

Para configurar os links basta clicar em Adicionar Link no Dashboard e um link sera gerado. Para usa-lo para enviar dados utilize programas de linux embarcado como o curl:
curl http://localhost:8000/dashboard/api?key=IePLlsKFO8mzHIiw --data "dado=49" 

Os dados serao armazenados em uma tabela para mais tarde poderem ser analizados

Graficos:

Aqui temos um grafico gerado com os dados dos links, inicialmente nos limitamos apenas à graficos de temperatura, inicialmente.
